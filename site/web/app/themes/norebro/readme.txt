Norebro is flexible and high performance multipurpose WordPress Theme for personal and corporate use. It comes with the most popular WordPress page builder WPBakery Page Builder (formerly Visual Composer). Create gorgeous pages and layouts fast and easy without coding.

Need help? Visit our support portal: https://colabrio.ticksy.com/


== Installation ==
Manual installation:

1. Upload the `norebro` folder to the `/wp-content/themes/` directory

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Theme Settings to change theme specific options
